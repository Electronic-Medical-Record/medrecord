# MedRecord - Política de Privacidade

Este diretório contém a página de política de privacidade do aplicativo MedRecord, hospedada no GitHub Pages.

## Configuração do GitHub Pages

1. Acesse: **Settings > Pages** no repositório
2. Em "Source", selecione: **Deploy from a branch**
3. Em "Branch", selecione: **main** e pasta **/docs**
4. Clique em **Save**

A página estará disponível em:
```
https://medrecords-organization.github.io/medrecord-app/
```

## Arquivos

- `index.html` - Página principal com a política de privacidade
- `icon.png` - Logo do aplicativo
- `_config.yml` - Configuração do Jekyll (GitHub Pages)

## Layout

A página usa o mesmo esquema de cores do aplicativo:
- Gradiente azul: `#0F172A` → `#1E3A8A`
- Destaque: `#60A5FA`
- Responsivo (mobile, tablet e desktop)

## Atualização

Para atualizar a política de privacidade, edite o arquivo `index.html` e faça commit/push para o GitHub.
